# Website Structure and Sitemap

## Pages
1. **Home/Bio** (index.html)
   - Hero section with name and tagline
   - Brief professional introduction
   - Professional photo (placeholder)
   - Navigation menu

2. **Work** (work.html)
   - Professional experience
   - Projects
   - Achievements
   - Skills

3. **Hobbies** (hobbies.html)
   - Personal interests
   - Activities
   - Photos/gallery of hobbies

4. **Articles** (articles.html)
   - List of articles with dates
   - Brief descriptions
   - Links to full articles

5. **Portfolio** (portfolio.html)
   - Gallery of work samples
   - Project descriptions
   - Images with captions

6. **Contact** (contact.html)
   - Contact information
   - Contact form
   - Social media links

## Navigation
- Clean horizontal navigation bar at the top
- Mobile-responsive hamburger menu for smaller screens
- Logo/name that links back to home page

## Design Elements
- **Color Scheme**: Black and white with minimal accent colors
- **Typography**: Clean, readable fonts (similar to Haidt's site)
- **Layout**: Minimalist, content-focused design
- **Responsive**: Adapts to all screen sizes (desktop, tablet, mobile)

## Technical Structure
- HTML5 semantic markup
- CSS3 with flexbox/grid for layouts
- Minimal JavaScript for interactive elements
- Responsive images
- Fast loading times
